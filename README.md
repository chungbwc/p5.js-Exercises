# p5.js Exercises



Exercises with [p5.js](https://p5js.org/) for media art/design.



This is the software repository for my creative coding exercises with the p5.js (Javascript) programming environment. The focus is in the Media Art and Design areas with the common libraries of computer vision, machine learning, interaction design. etc. It will cover the integration with [OpenCV](https://opencv.org/), [MediaPipe](https://developers.google.com/edge/mediapipe/solutions/guide), [Yolo models](https://docs.ultralytics.com/).
